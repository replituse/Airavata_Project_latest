## Packages
leaflet | Interactive maps for Dam locations
react-leaflet | React components for Leaflet maps
recharts | Charting library for simulation results
file-saver | Client-side file saving for INP downloads
@types/leaflet | TypeScript definitions for Leaflet
@types/file-saver | TypeScript definitions for file-saver
clsx | Utility for constructing className strings conditionally
tailwind-merge | Utility for merging Tailwind CSS classes

## Notes
Leaflet CSS must be imported in global styles.
Map requires a height to be visible.
Simulation results are mocked in the backend initially.
The "Builder" uses dynamic form fields based on the selected element type.
